# LND Brisitas

Proyecto de tienda web para la asignatura de Lenguaje de Marcas. Incluye catalogo, carrito, checkout simulado, contacto y panel admin local.

## Autor
- Eudaldo Alvaro Cal Saul

## Estado
- Proyecto final de curso (front-end sin backend real)

## Caracteristicas
- Catalogo por categorias
- Carrito persistente con localStorage
- Checkout simulado (sin pasarela real)
- Panel admin para agregar productos locales
- Paginas informativas legales y blog

## Demo local
1. Abrir el archivo `index.html` en el navegador.
2. Agregar productos al carrito.
3. Finalizar compra en `checkout.html`.
4. Gestionar productos en `admin.html`.

## Estructura
- `index.html`: Home
- `admin.html`: Panel admin
- `checkout.html`: Checkout
- `contacto.html`: Contacto
- `styles.css`: Estilos globales
- `script.js`: Logica principal
- `docs/`: Documentacion tecnica
