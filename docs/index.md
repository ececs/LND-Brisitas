## Resumen
Proyecto de tienda web con HTML, CSS y JavaScript (vanilla). Incluye catalogo por categorias, carrito persistente, checkout simulado, contacto y panel admin local.

## Alcance
- Sitio estatico sin backend real.
- Persistencia local con localStorage.
- Enfoque en buenas practicas de maquetacion y navegacion.
