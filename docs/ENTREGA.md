# Entrega academica

## Asignatura
- Lenguaje de Marcas

## Objetivo
Desarrollar una tienda web funcional con HTML, CSS y JavaScript basico, incluyendo paginas informativas y flujos de compra.

## Alcance
- Catalogo por categorias
- Carrito persistente
- Checkout simulado
- Contacto
- Panel admin local

## Tecnologias
- HTML5
- CSS3
- JavaScript (vanilla)

## Observaciones
- Proyecto educativo, sin backend real.
